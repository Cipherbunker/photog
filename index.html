Turn your Pictures folder into a Photography website!


(source: www.superformosa.nl)

Getting started

Installing Photog! is super easy thanks to Cpanminus and local::lib. Simply run the following command: 

$ cpanm https://github.com/rtts/photog/tarball/stable

Also, make sure you have ImageMagick installed before you start generating websites. More detailed installation instruction can be found in the readme file.

Creating a website

To generate a photography website using the default settings, simply cd to your Pictures directory and execute photog with the destination directory as its argument: 

cd ~/Pictures
photog ~/public_html

Your website's main page should now be available in the destination directory. Go have a look!

Customization

The output of Photog! be customized in a number of ways. Please read the manpage which contains all the documentation.
