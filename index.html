Turn your Pictures folder into a Photography website!


(source: www.superformosa.nl)

  • Free and Open Source

    The source code is freely available under the terms of the GPL.

  • For Linux, Mac and Windows

    It works on every operating system that has Perl and ImageMagick installed.

  • Fully automated

    Generates image thumbnails, watermarked images, album previews, and all the needed HTML files.

  • Intelligent updating

    Keeps the photography website up-to-date by only regenerating the files that changed.

  • Exif aware

    Pictures are sorted in the order they were actually photographed.

  • Pleasing to the eye

    The default template packs images into horizontal rows. When you click on a thumbnail, it seamlessly zooms to full screen.

  • Highly configurable

    Features an intuitive and powerful configuration system, where albums inherit settings from their parent album.

  • Works out of the box

    To create a photography website you only have to type “photog” at the command line.

Installation

Installing Photog! is super easy thanks to Cpanminus and local::lib. The following command will automatically install the latest stable version: 

$ cpanm https://github.com/rtts/photog/tarball/stable

Also, make sure you have ImageMagick installed before you start generating websites. More detailed installation instruction can be found in the readme file.

Creating a website

To generate a photography website using the default settings, simply cd to your Pictures directory and execute photog with the destination directory as its argument: 

cd ~/Pictures
photog ~/public_html

Your photography website should now be available in the destination directory. Go have a look!

Documentation

The complete documentation is available as a manual page after installation. Execute the command man photog to read it. Another manual page, man Photography::Website, documents the Perl library that is used by Photog! For your convenience, both manual pages are also reprinted below.
